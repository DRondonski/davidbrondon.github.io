Ice Accretion Research

Airfoil icing research using NASA LEWICE, OpenFOAM, ParaView, and a cross-platform workflow.

Overview

This project focused on airfoil ice accretion modeling using a workflow built around NASA LEWICE, OpenFOAM, and ParaView. The work involved more than analysis alone: a big part of the contribution was making the software stack usable and dependable for the rest of the team.

Core Contribution

I led the effort to get NASA LEWICE working on macOS and connected that tooling with the geometry and CFD pipeline the group was already building. That meant handling setup friction, data transfer between tools, and post-processing in a way that others could actually reuse.

Workflow

  • Prepare airfoil geometries for analysis
  • Run icing predictions with LEWICE
  • Push usable geometry into OpenFOAM
  • Visualize flow and accretion behavior in ParaView
  • Organize plots and comparisons for reporting and presentation

Outcome

The result was a more stable and usable icing-analysis workflow, plus conference-ready research output that tied together computation, visualization, and communication.
